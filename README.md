# Ping-Pong на FPGA

Учебный проект по реализации классической игры Pong на FPGA с выводом изображения по HDMI.

Проект выполнили студенты группы `8В32`:

- Крамаренко
- Чарков
- Быков

В рамках проекта была разработана аппаратная версия игры для двух игроков. Логика игры, формирование видеосигнала, работа с UART и настройка HDMI-передатчика реализованы на `VHDL`.


## Что умеет проект

- запускает игру Ping-Pong для двух игроков;
- выводит изображение по `HDMI`;
- поддерживает несколько разрешений экрана;
- позволяет менять скорость мяча;
- принимает команды по `UART`;
- содержит тестбенчи для проверки основных модулей.

## Поддерживаемые режимы

Проект поддерживает следующие видеорежимы:

- `640x480 @ 60 Гц`
- `800x600 @ 60 Гц`
- `1024x768 @ 60 Гц`

## Основные модули проекта

- `pongGame.vhd`  
  Главный модуль проекта. В нём объединяются все остальные блоки: генерация тактов, видеовывод, игровая логика, UART и инициализация HDMI.

- `pongSync.vhd`  
  Формирует сигналы синхронизации изображения: `HSYNC`, `VSYNC`, `DE`, а также координаты текущего пикселя.

- `pongImg.vhd`  
  Отвечает за игровую часть: движение ракеток, движение мяча, границы поля, счёт и отображение объектов на экране.

- `cmd_reset_parser.vhd`  
  Обрабатывает команды, приходящие по `UART`.

- `uart_rx.vhd`  
  Модуль приёма данных по `UART`.

- `uart_tx.vhd`  
  Модуль передачи данных по `UART`.

- `adv7513_setup.sv`  
  Выполняет настройку HDMI-передатчика `ADV7513` по `I2C`.

- `adv7513_wrapper.vhd`  
  Обёртка для подключения и запуска настройки `ADV7513`.

## Управление

Для управления игроками используются отдельные кнопки:

- `lbutton_p1` `KEY[1] `, `rbutton_p1` `KEY[2]` — управление первым игроком
- `lbutton_p2` `KEY[3]`, `rbutton_p2` `KEY[4]` — управление вторым игроком

Сигнал `KEY[0]` используется для сброса игровой логики.

## Команды UART

Проект поддерживает следующие команды:

- `RESET` — перезапуск игры
- `RES0` — установить разрешение `640x480`
- `RES1` — установить разрешение `800x600`
- `RES2` — установить разрешение `1024x768`
- `SPEED1` — минимальная скорость мяча
- `SPEED2`
- `SPEED3` — скорость по умолчанию
- `SPEED4`
- `SPEED5` — максимальная скорость мяча

Примеры ответов:

- `OK`
- `OK 640x480`
- `OK 800x600`
- `OK 1024x768`
- `OK SPEED1`
- `OK SPEED2`
- `OK SPEED3`
- `OK SPEED4`
- `OK SPEED5`
- `Unknown command`
